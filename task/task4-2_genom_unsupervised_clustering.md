# Учебное задание 4.2: Обучение без учителя - кластеризация белковых последовательностей с использованием гиперболических эмбеддингов

## Задача

> **У вас есть датасет** белковых последовательностей в формате FASTA, содержащий аминокислотные последовательности от 24 различных видов. Каждая последовательность помечена идентификатором вида, но **вы не должны использовать эти метки при обучении** — это *обучение без учителя*.

> **Вы должны написать код для кластеризации белковых последовательностей** в значимые группы, выполнив следующие шаги:
> 1. Получить векторные эмбединги аминокислотных последовательностей из датасета `task4_2-genomic-amino-acid-code.fasta``
> 2. Использовать **гиперболическое пространство** (модель шара Пуанкаре) для лучшего отражения иерархических связей
> 3. Построить дендрограмму полученных эмбедингов
> 4. Оценить качество кластеризации

** Ожидаемый результат**: решение для кластеризации, которое группирует белковые последовательности в близкие ветви дендрограммы *без использования меток видов в процессе обучения*.

---

## Описание датасета

| Свойство | Значение |
|----------|-------|
| **Формат** | FASTA (`.fasta`) |
| **Образцы** | 24 белковые последовательности |
| **Признаки** | Строки аминокислот (20 стандартных остатков + неизвестные) |
| **Метки** | Названия видов (только для оценки) |
| **Файл** | `task4_2-genomic-amino-acid-code.fasta` |

Пример:
```
>Human_TP53
MEEPQSDPSVEPPLSQETFSDLWKLLPENNVLSPLPSQAM...
>Zebrafish_TP53
MEEPQSDPSVEPPLSQETFSDLWKLLPENNVLSPLPSQAM...
```
